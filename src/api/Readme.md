# Requirements
- Go compiler
- Nodemon (Optional - used for live reloading)

# How to Run
- Step 1: Install Dependencies `go mod download`
- Step 2: `go run .` or using Nodemon `./hotreload.bat`
- Step 3: Open the `controllers` folder to find all the defined routes
